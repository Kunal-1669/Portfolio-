Retrieval Augmented Generation (RAG) Pipeline

April 2023 | Python, AWS

Developed a ready-to-deploy RAG pipeline for efficient document retrieval from large repositories, leveraging LangChain, Chroma vector database, and large language models like Llama and Mixtral. Implemented multi-vector retrieval, contextual text chunking, and PDF parsing using libraries like Unstructured.io, enabling seamless information extraction from extensive document collections.

News Article Summarization

November 2023 | AWS and Python
